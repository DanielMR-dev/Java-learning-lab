# Java Learning Lab

Repositorio personal de aprendizaje y práctica profesional en Java.

Este repositorio documenta mi progreso aprendiendo Java moderno, programación orientada a objetos, colecciones, testing, Maven, Spring Boot y desarrollo backend.

## Objetivo

Construir una base sólida en Java y desarrollar proyectos prácticos orientados a empleabilidad como desarrollador backend Java.

## Stack principal

- Java 25
- Maven
- JUnit
- IntelliJ IDEA
- Git y GitHub
- Spring Boot
- PostgreSQL
- Docker

## Estructura

| Carpeta                          | Contenido                                                |
| -------------------------------- | -------------------------------------------------------- |
| `01-java-fundamentals`           | Sintaxis, variables, condicionales, ciclos, métodos      |
| `02-object-oriented-programming` | Clases, objetos, encapsulamiento, herencia, polimorfismo |
| `03-collections-generics`        | List, Set, Map, Queue, generics                          |
| `04-exceptions-files`            | Excepciones, manejo de archivos, java.io, java.nio       |
| `05-lambdas-streams`             | Lambdas, functional interfaces, Stream API               |
| `06-testing-junit`               | Pruebas unitarias con JUnit                              |
| `07-maven-projects`              | Proyectos Java con Maven                                 |
| `08-spring-boot`                 | APIs REST con Spring Boot                                |
| `projects`                       | Proyectos completos de portafolio                        |
