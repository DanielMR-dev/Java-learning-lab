# Semana 3 — Condicionales, ciclos y métodos

## Tema

En esta semana estudié estructuras de control de flujo en Java, ciclos y métodos. El objetivo fue aprender a tomar decisiones, repetir instrucciones y organizar el código en bloques reutilizables.

## Qué aprendí

Aprendí a usar `if`, `else` y `else if` para ejecutar código dependiendo de una condición.

También aprendí a usar `switch` para manejar múltiples opciones de una forma más ordenada, especialmente en aplicaciones basadas en menús.

Practiqué ciclos con `for`, `while` y `do while`. El ciclo `for` es útil cuando conozco la cantidad de repeticiones. El ciclo `while` es útil cuando la repetición depende de una condición. El ciclo `do while` es útil cuando necesito ejecutar el bloque al menos una vez.

Además, aprendí a crear métodos para dividir el programa en partes más pequeñas, legibles y reutilizables. También estudié parámetros, argumentos, retorno de valores y sobrecarga de métodos.

## Conceptos importantes

Una condición es una expresión que produce un valor booleano: `true` o `false`.

Un método es un bloque de código con nombre que realiza una tarea específica.

Un parámetro es una variable definida en la firma de un método.

Un argumento es el valor real que se envía al llamar un método.

El retorno de un método es el valor que el método devuelve al lugar donde fue llamado.

La sobrecarga de métodos permite crear varios métodos con el mismo nombre, siempre que tengan diferentes parámetros.

## Mini proyecto

### NumberGuessingGame

El mini proyecto consiste en un juego de adivinar un número aleatorio.

Características principales:

* Genera un número aleatorio.
* Define un máximo de intentos.
* Muestra pistas al usuario.
* Indica si el número ingresado es muy alto o muy bajo.
* Permite jugar de nuevo.
* Separa la lógica en métodos.
